Web Development

Probably the most interesting module for me as even with little amount of programming knowledge you can see visual result of coding in a form of web page. I learned how to markup a webpage with HTML and how to style it with CSS.

For styling of my first semester project we used W3CSS - a framework that helped me to make the website attractive according to current styling standards. You can see the website in Portfolio section.

In second semester I learned basic JavaScript. This gave me the opportunity to process inputs from user and output a result dynamically to a web page. Understanding and coding in JavaScript is incredibly important as all the server-side applications and backend of websites is programmed with JavaScript based frameworks such as Node.js and Angular.js
