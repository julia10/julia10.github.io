Agile Methodologies

On this module I learned what is software engineering as a concept, what is the role of software engineer and what are my ethical and professional responsibilities. I understood such methodologies as Scrum, Extreme programming, Feature driven development and Kanban.

This module introduces students to the concept of software engineering as well as describing the role of software engineers, including their ethical/professional responsibilities. It provides an introduction to the following different types of agile methodologies: Scrum, Extreme Programming, Feature Driven Development and Kanban by focusing on the key processes of Requirements, Pair Programming, Testing, and Configuration Management.

In this module we are learning:

  • How to interact with coworkers in Software development teams

  • Code of Ethics in development and how it should be implemented by us as software developers in the future

  • Software testing process and how to write User Stories

  • Test driven development

  • Agile processes

  • We were practicing group work while preparing a presentation about Code of Ethics problems. The topic of presentation was Equifax Data breach and how it could have been avoided if their company valued the principles of Ethics
